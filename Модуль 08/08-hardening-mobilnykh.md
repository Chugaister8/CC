# 8.8. Hardening мобільних пристроїв

Мобільний пристрій із заводськими налаштуваннями — це не безпечний пристрій, а лише зручний. Виробники встановлюють баланс між зручністю і безпекою, що схиляється до зручності: більше дозволів за замовчуванням, менше запитів — вища оцінка у магазинах. Hardening — свідоме зміщення цього балансу у бік безпеки через конкретні налаштування.

> 📖 Ключові терміни — у [глосарії модуля](00-glosariy.md).

## iOS Hardening: покроковий чек-лист

### Основні налаштування безпеки

```
Налаштування → Face ID / Touch ID і код-пароль:
☐ Код: увімкнено
☐ Код-пароль: буквено-цифровий (не 4 і не 6 цифр — повноцінний пароль)
☐ Вимагати код: Негайно
☐ Стерти дані: Enabled (після 10 невдалих спроб)

Налаштування → Конфіденційність і безпека:
☐ Трекінг: "Allow Apps to Request to Track" → Off
☐ Локальна мережа: перегляньте список — видаліть зайві
☐ Bluetooth: лише застосунки, що явно потребують

Налаштування → Загальні → VPN і управління пристроєм:
☐ Перегляньте встановлені профілі (не повинно бути невідомих)
```

### Захист від фізичного доступу

```
Налаштування → Face ID і код-пароль:
☐ Вимкніть "Allow Access When Locked":
  - Control Center: Off
  - Notification Centre: Off  
  - Siri: Off (Siri може витікати дані без розблокування)
  - Wallet: Off (якщо не потрібен)
  - Reply with Message: Off
  - Return Missed Calls: Off

Налаштування → Siri та пошук:
☐ "Allow Siri When Locked": Off
```

### Конфіденційність геолокації

```
Налаштування → Конфіденційність → Служби геолокації:
☐ Для кожного застосунку:
  - "Ніколи" — якщо геолокація не потрібна
  - "Під час використання" — якщо потрібна тільки при відкритому застосунку
  - "Завжди" — ТІЛЬКИ для Maps, Find My, Fitness трекерів

☐ Системні служби → прокрутіть вниз:
  - "Важливі місця": Off (Apple зберігає місця де ви буваєте)
  - iPhone Analytics: Off
  - Routing & Traffic: Off (якщо хочете конфіденційності)
```

### Мережева безпека

```
Налаштування → Wi-Fi → "Попросити підключення": Notify (не Auto)
☐ Приватна адреса Wi-Fi: увімкнено (ротація MAC-адреси)
☐ Обмежте Wi-Fi мережі, що автоматично підключаються
☐ Вимкніть Wi-Fi Assist (автоматичне переключення на стільникову — може обійти VPN)

Налаштування → Bluetooth:
☐ Вимикати Bluetooth у публічних місцях (Bluetooth tracking, AirTag stalking)
```

## Android Hardening: покроковий чек-лист

### Блокування екрану і шифрування

```
Налаштування → Безпека → Блокування екрана:
☐ PIN: мінімум 6 цифр (краще: буквено-цифровий пароль)
☐ Smart Lock: вимкнути або мінімізувати trusted places/devices

Налаштування → Безпека → Шифрування:
☐ Encrypt phone: Enabled (Android 6+ шифрує автоматично якщо є PIN)
```

### Дозволи застосунків

```
Налаштування → Конфіденційність → Менеджер дозволів:
☐ Мікрофон: перегляньте, видаліть зайві дозволи "Завжди"
☐ Камера: те саме
☐ Геолокація: "Лише під час використання" замість "Завжди"
☐ Читання SMS: тільки диктофон/банківські застосунки
☐ Контакти: мінімізуйте доступ

Налаштування → Конфіденційність:
☐ Permission auto-reset: On (Android 11+)
☐ Show clipboard access: On
☐ Camera/Mic access: On (індикатор у статус-барі)
```

### Захист від установки з невідомих джерел

```
Налаштування → Застосунки → "Спеціальний доступ застосунків":
☐ Установка невідомих застосунків: вимкнено для ВСІХ застосунків
   (або дозволено лише для конкретного файлового менеджера якщо потрібно)
```

### Google і Huawei специфічні налаштування

```
Налаштування → Google → Manage your Google Account → Data & Privacy:
☐ Web & App Activity: Off або Auto-delete (3 months)
☐ Location History: Off
☐ YouTube History: Off або Auto-delete

Google Play Protect (Налаштування → Security → Google Play Protect):
☐ Scan apps with Play Protect: On
☐ Improve harmful app detection: On
```

## Корпоративний профіль і Work Profile (Android Enterprise)

**Android Enterprise Work Profile** розділяє корпоративні і особисті застосунки на одному пристрої (BYOD):

```
Personal Profile:                    Work Profile (підключений до MDM):
├── Особисті застосунки              ├── Корпоративна пошта
├── Особисті фото                    ├── Teams/Slack корпоративний
├── Особистий браузер                ├── Корпоративні документи
└── (MDM не бачить)                  └── (MDM керує)
```

**Переваги:**
- IT не має доступу до особистих даних.
- Корпоративні дані ізольовані — при звільненні MDM видаляє лише Work Profile.
- Окремий пароль для Work Profile.

**Налаштування Android Work Profile:**
```
Якщо вашій організації є Intune або Workspace ONE:
1. Встановіть Microsoft Intune Company Portal (або відповідний)
2. Дотримуйтесь інструкції щодо реєстрації
3. Система автоматично створить Work Profile
```

## VPN: мобільна безпека

**Коли VPN обов'язковий на мобільному:**
- Будь-яка публічна Wi-Fi мережа (кав'ярня, аеропорт, готель, конференція).
- Підключення до корпоративних ресурсів поза офісом.
- Країни з обмеженим інтернетом або активним перехопленням.

**Вибір VPN-протоколу:**

| Протокол | Безпека | Швидкість | Мобільний |
|---|---|---|---|
| **WireGuard** | ✅ Висока | ✅ Висока | ✅ Відмінно |
| **IKEv2/IPSec** | ✅ Висока | ✅ Висока | ✅ Вбудований у iOS/Android |
| **OpenVPN** | ✅ Висока | Середня | Потрібен застосунок |
| L2TP/IPSec | Слабка | — | ❌ Не рекомендується |
| PPTP | ❌ Зламаний | — | ❌ Ніколи |

**Налаштування VPN на iOS:**
```
Налаштування → VPN і управління пристроєм → VPN → Додати конфігурацію VPN
Тип: IKEv2 або WireGuard
```

**Kill Switch для мобільного VPN:**
```
Android 8+: Налаштування → Мережа → Розширені налаштування:
☐ VPN → Завжди ввімкнений VPN: On
☐ Блокувати з'єднання без VPN: On
```

## Поведінкові практики безпеки

Технічні налаштування доповнюються усвідомленою поведінкою:

**При поїздках:**
- Перетинаючи кордон до країни з ризиком примусового огляду пристрою (Росія, Білорусь, Китай) — рекомендується «чистий» пристрій.
- Увімкніть Airplane mode на кордоні якщо не використовуєте.
- Вимкніть Face ID / Touch ID якщо є ризик примусового розблокування (у деяких країнах поліція може вимагати притулити палець) — залиште лише пін.

**Публічні зарядки:**
```
Використовуйте USB Data Blocker (PortaPow, eSynce) при зарядці
у публічних USB-портах. Або заряджайтесь через власний адаптер до розетки.
```

**App hygiene:**
- Видаляйте застосунки, якими не користуєтесь (менше поверхня атаки).
- Перевіряйте оновлення щотижня — або увімкніть автоматичні.
- Перед встановленням нового застосунку — перегляньте запитувані дозволи.

## Міні-вправа

Проведіть «аудит безпеки» свого смартфона прямо зараз:

```
Чек-лист за 10 хвилин:
☐ Якого типу блокування екрана? (Менше 6 цифр — небезпечно)
☐ Останнє оновлення безпеки? (Старіше 3 місяців — ризик)
☐ Скільки застосунків мають доступ до мікрофона?
☐ Скільки застосунків мають доступ "Завжди" до геолокації?
☐ Чи є Meeting застосунки (Zoom, Teams) з доступом "Завжди"?
☐ Чи увімкнено "Встановлення з невідомих джерел"?
☐ Чи є незнайомі профілі конфігурації (iOS)?
☐ Чи є VPN і коли ви його використовуєте?
```
