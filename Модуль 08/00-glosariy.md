# Глосарій модуля 08

Усі ключові терміни модуля «Безпека мобільних пристроїв та IoT» за алфавітом. Українські терміни — за українською абеткою; англійські — в блоці A-Z.

## А

**Актуатор (Actuator)** — пристрій, що перетворює сигнал у фізичну дію; компонент рівня Perception у IoT. *(8.5)*

**Air Gap** — фізична ізоляція мережі від інтернету; обходиться через USB або компрометацію персоналу. *(8.7)*

**Автентифікація пристрою (Device Authentication)** — підтвердження ідентичності IoT через X.509 або PSK. *(8.6)*

## В

**VLAN (Virtual LAN)** — логічне розділення мережі; IoT у окремому VLAN ізольовані від корпоративних систем. *(8.9)*

**Вбудована система (Embedded System)** — комп'ютерна система з обмеженими ресурсами для конкретного завдання; основа IoT. *(8.5)*

## Д

**Дозвіл (Permission)** — явна згода користувача на доступ застосунку до ресурсу (камера, мікрофон, геолокація). *(8.1, 8.3)*

**DCS (Distributed Control System)** — розподілена система управління для комплексних виробничих процесів. *(8.7)*

## З

**Застосунок-шпигун (Stalkerware)** — ПЗ для прихованого стеження без відома власника пристрою. *(8.2)*

## І

**ICS (Industrial Control Systems)** — загальний термін для промислових систем управління: SCADA, DCS, PLC. *(8.7)*

**IT/OT конвергенція** — інтеграція корпоративних IT і промислових OT мереж; розширює attack surface. *(8.7)*

## К

**Крипто-стирання (Crypto Erase)** — знищення даних через видалення ключа шифрування; ефективне для flash-пам'яті. *(8.6)*

## М

**MDM (Mobile Device Management)** — централізоване управління мобільними пристроями: конфігурація, застосунки, Remote Wipe. *(8.4)*

**MAM (Mobile Application Management)** — управління лише корпоративними застосунками і даними; для BYOD. *(8.4)*

**MCU (Microcontroller Unit)** — мікроконтролер з CPU, RAM, Flash на одному чипі; основа IoT. *(8.5)*

## О

**OT (Operational Technology)** — ПЗ і обладнання, що контролює фізичні процеси; протилежність IT. *(8.7)*

**OTA (Over-the-Air)** — бездротове оновлення прошивки без фізичного підключення. *(8.6)*

## П

**PLC (Programmable Logic Controller)** — промисловий мікроконтролер для автоматизації конкретних задач. *(8.7)*

**Прошивка (Firmware)** — вбудоване ПЗ IoT у flash-пам'яті; потребує регулярних оновлень. *(8.6)*

**Purdue Model** — ієрархічна архітектура ізоляції IT і OT мереж (ISA-95); 5 рівнів. *(8.7)*

## Р

**Root (Android)** — отримання привілеїв суперкористувача; порушує ізоляцію застосунків. *(8.2)*

**RASP (Runtime Application Self-Protection)** — захист мобільного застосунку під час виконання. *(8.3)*

## С

**Sandbox** — ізольоване середовище виконання; застосунок не має доступу до ресурсів інших. *(8.1)*

**SCADA** — система диспетчерського управління; управляє розподіленою інфраструктурою. *(8.7)*

**SELinux** — Mandatory Access Control у ядрі Linux; обмежує дії навіть root-процесів в Android. *(8.1)*

**SS7** — телекомунікаційний протокол 1975 року з вразливостями; дозволяє перехоплення SMS. *(8.2)*

**SIM Swapping** — соціальна інженерія оператора для перенесення номера на нову SIM. *(8.2)*

**Сенсор (Sensor)** — пристрій, що перетворює фізичну величину в цифровий сигнал. *(8.5)*

## Т

**TEE (Trusted Execution Environment)** — захищена область виконання в ARM TrustZone; ізольована від основної ОС. *(8.1)*

## У

**UEM (Unified Endpoint Management)** — управління всіма кінцевими пристроями: мобільні, ноутбуки, IoT. *(8.4)*

## Ф

**Firmware Signing** — криптографічний підпис прошивки; захист від підміни при оновленні. *(8.6)*

## A-Z (англійські абревіатури та назви)

**BYOD (Bring Your Own Device)** — власні пристрої для роботи; потребує MAM без повного MDM. *(8.4)*

**Certificate Pinning** — перевірка конкретного сертифіката сервера; захист від MITM. *(8.3)*

**CoAP (Constrained Application Protocol)** — REST-подібний протокол на UDP для ресурсообмежених IoT. *(8.5)*

**COPE (Corporate-Owned, Personally Enabled)** — корпоративний пристрій з дозволом особистого використання. *(8.4)*

**Evil Twin** — підроблена Wi-Fi точка доступу з тим самим SSID; жертва потрапляє під MITM. *(8.2)*

**FBE (File-Based Encryption)** — Android шифрування кожного файлу окремим ключем. *(8.1)*

**HMI (Human-Machine Interface)** — панель оператора для управління промисловим обладнанням. *(8.7)*

**Industroyer** — кіберзброя, що атакувала українську електромережу у 2016 і 2022 роках. *(8.7)*

**Jailbreak** — обхід обмежень iOS на виконання непідписаного коду; порушує Sandbox. *(8.2)*

**Juice Jacking** — атака через заражений USB-зарядний порт; захист: USB Data Blocker. *(8.2)*

**LoRaWAN** — LPWAN протокол для IoT з дальністю до 15 км; шифрування AES-128. *(8.5)*

**LPWAN** — сімейство протоколів для IoT з великою дальністю і малим споживанням. *(8.5)*

**MASVS (Mobile Application Security Verification Standard)** — стандарт OWASP для аудиту мобільних застосунків. *(8.3)*

**Mirai** — ботнет 2016 з 600 000+ IoT пристроїв через дефолтні паролі; DDoS 1.2 Tbps. *(8.5, 8.6)*

**Modbus** — промисловий протокол 1979 без автентифікації і шифрування; найпоширеніший OT протокол. *(8.7)*

**MQTT** — легкий pub/sub протокол IoT; порт 1883 (без TLS) або 8883 (з TLS). *(8.5)*

**MTD (Mobile Threat Defense)** — аналіз загроз на пристрої в реальному часі; виявляє Evil Twin, malware. *(8.4)*

**NAC (Network Access Control)** — контроль доступу пристроїв; авто-розміщення у правильний VLAN. *(8.9)*

**OPC-UA** — сучасний промисловий протокол з TLS і X.509; рекомендований для нових OT. *(8.7)*

**Overlay Attack** — відображення підробленого екрану поверх легітимного застосунку. *(8.3)*

**Pegasus** — комерційний spyware від NSO Group; zero-click вразливості iOS і Android. *(8.2)*

**RTU (Remote Terminal Unit)** — польовий контролер для SCADA; збирає дані з сенсорів. *(8.7)*

**Secure Boot** — верифікація компонентів завантаження через Chain of Trust. *(8.6)*

**Secure Element (SE)** — захищений мікросхем для зберігання ключів; аналог Secure Enclave. *(8.6)*

**Secure Enclave** — окремий процесор у Apple Silicon для ключів, біометрії і платежів. *(8.1)*

**Shodan** — пошукова система для підключених до інтернету пристроїв. *(8.5, 8.9)*

**Stuxnet** — перша кіберзброя, що фізично пошкодила обладнання; атакувала іранські центрифуги. *(8.7)*

**TrustZone** — апаратна ізоляція ARM: Secure World і Normal World; основа Android Keystore. *(8.1)*

**USB Data Blocker** — адаптер, що пропускає лише живлення і блокує дата-лінії. *(8.8)*

**Verified Boot** — перевірка цілісності Android розділів через dm-verity. *(8.1)*

**Work Profile** — ізольований контейнер Android для корпоративних застосунків на BYOD. *(8.4)*

**Zero-Touch Enrollment** — Apple DEP: автоматична MDM-реєстрація при першому увімкненні. *(8.4)*

**Zigbee** — mesh протокол 2.4 GHz для розумного дому (IEEE 802.15.4). *(8.5)*

**Z-Wave** — пропрієтарний mesh протокол 868 MHz для розумного дому. *(8.5)*
