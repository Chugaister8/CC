# Модуль 08. Безпека мобільних пристроїв та IoT

> Смартфон у кишені — це комп'ютер потужніший за серверну кімнату 2000-х, що постійно підключений до інтернету, знає вашу геолокацію, має доступ до корпоративної пошти і зберігає фінансові дані. При цьому більшість людей витрачають на налаштування його безпеки нуль хвилин. Те саме стосується IoT: камера спостереження з дефолтним паролем, роутер без оновлень, «розумний» промисловий контролер з прямим підключенням до інтернету — мільярди пристроїв, що стали частиною корпоративних і державних мереж без будь-якого security review. Цей модуль — про те, як захистити ці два найбільш ігнорованих класи пристроїв.

## Цілі модуля

Після опрацювання модуля ви зможете:

- **Запам'ятати:** OWASP Mobile Top 10; компоненти MDM/MAM/UEM; ключові протоколи IoT (MQTT, CoAP, Zigbee, Z-Wave); відмінності між IT і OT безпекою.
- **Зрозуміти:** пояснити архітектурні відмінності безпеки Android і iOS; описати модель загроз для IoT-пристрою; пояснити, чому SCADA-системи є критичною інфраструктурою з унікальними вимогами до безпеки.
- **Застосувати:** провести hardening мобільного пристрою; виявити небезпечні IoT-пристрої в мережі; налаштувати VLAN-сегментацію для IoT; скласти чек-лист безпеки мобільних застосунків.

**Орієнтовна тривалість:** 10–12 годин.
